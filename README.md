# YML2TMX 

Paradox Interactive社のゲームエンジンClausewitzのローカライズファイル2つを統合し
OmegaTで読み込み可能な翻訳メモリを生成します。

2つのymlファイルから、OmegaTのプロジェクトに配置することで使用できる
翻訳メモリ(tm)もしくは用語集(glossary)を生成します。


## インストール

ZIPファイルを解凍し、任意の場所に置きます。
